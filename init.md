# What is Routines
Routines is an mobile app that should help users create and run routines.

## Basic Features
1. Home screen
  a. name of the app on top as header saying routines
  b. below that a quote that comes up as if each letter is being types.( we will create a list of 100 quotes and cycle through them evertime user reloads or opens the app)
  c. list of routine that the users
  d. on the bottom right - create routine button which will open up screen 2

2. Create routine screen
  a. name of the routine
  b. task name, duration of task ( time setter as well as pills), task icon ( we will maintain a list of icons)
  c. add a new task
  d. ability of re-oder the tasks
  e. save task
