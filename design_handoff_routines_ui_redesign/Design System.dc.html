Design Rules
Card elevation: always 2dp shadow. Never use Material tonal surface colors — use CardWhite + shadow instead.
Tinted surfaces (#F7EDE5) are for content surfaces (quote, icon circles, secondary buttons). Never use them for interactive cards.
All section headers follow the same pattern: ExtraBold 10sp uppercase + 2px NearBlack bottom rule + count badge aligned right.
Bottom sheets always: 28dp top radius, white bg, 40px drag handle centered, consistent padding 20dp horizontal.
No scaffold-provided top app bars. Every screen uses a custom nav bar: 36dp circular back/close button + Nunito ExtraBold title.
Dynamic color must be disabled. The app uses a fixed palette — wallpaper tinting would break the warm white + burnt orange system.
Metadata labels (duration, task count) always: SubText color, ExtraBold, 10sp, uppercase, 1.5sp letter-spacing.
